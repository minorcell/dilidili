# 更新日志

## v2.0.0 - 纯Go实现 (2025-06-15)

### 🎉 重大更新
- **移除ffmpeg依赖**: 完全使用纯Go实现音视频合并功能
- **简化部署**: 用户无需额外安装ffmpeg即可使用全部功能
- **提升性能**: 减少外部进程调用，提升合并速度

### 🔧 技术改进
- 实现了纯Go的MP4容器处理
- 优化了B站.m4s文件的合并逻辑
- 改进了错误处理和日志输出

### 📦 部署优势
- 单一可执行文件，无外部依赖
- 跨平台兼容性更好
- 安装包体积减少（移除76MB的ffmpeg）

### 🚀 使用方法
```bash
# 编译
go build -o dilidili .

# 运行
./dilidili
```

### 📋 技术细节
- 使用标准库的`encoding/binary`处理MP4格式
- 实现了ftyp box的标准化创建
- 智能跳过重复的文件头信息
- 保持与原版本完全相同的用户界面

### ⚠️ 注意事项
- 此版本专门针对B站的.m4s格式优化
- 如果遇到特殊格式问题，请反馈Issue
- 兼容性已在常见视频格式上测试
